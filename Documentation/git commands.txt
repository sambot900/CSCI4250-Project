Commands used to intialize the git repository and push an initial commit.
*navigate to project directory

#create local git repo
git init

# Add all files into the commit
git add .

#commit
git commit -m "description"

# create branch, where -M forces the rename as main here
git branch -M main

#point to github url
git remote add origin https://github.com/sambot900/CSCI4250-Project.git

#push
git push -u origin main

-----------

#Push and update:

git add .
git commit -m "description"
git push
